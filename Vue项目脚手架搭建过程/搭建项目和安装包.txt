预先准备安装环境
node(8+) + npm + vue-cli



1. 装包

	全局安装 `webpack` 和 `webpack-cli`

		npm i webpack webpack-cli -g

	全局安装 `vue-cli` 脚手架包

		npm i vue-cli -g

2. 使用脚手架命令新建一个`webpack`工程化的`vue`项目

	找一个风水宝地

		vue init webpack 项目名称(目录名称)

	例如:

		PS C:\Users\LTC\Desktop> vue init webpack vue-cms
